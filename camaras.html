Instalación y configuración de cámaras de seguridad

Se presentarán las instrucciones para instalar y configurar una cámara de seguridad, específicamente la cámara de la marca Hikvision, reconocida por sus productos de videovigilancia y su durabilidad. Las cámaras a utilizar provienen del kit "4 Channel HD, Wi-Fi Kit" de la misma marca.

Material

Este kit cuenta con los siguientes materiales:

  • 1 NVR Wi-Fi de 4 canales (Dispositivo especializado en videovigilancia IP para grabar y almacenar video.)
  • 4 cámaras tipo bullet de 2 MP.
  • Disco duro preinstalado de 1 TB.
  • Cable de red.
  • Cable HDMI.
  • 5 Enchufes de alimentación de 12 voltios.

Instrucciones

  1. Conexión del NVR y las cámaras: Conecta el NVR y las cámaras a sus respectivos enchufes de alimentación, luego conecta el NVR a un monitor o televisor. (Para mejor uso de las cámaras, conecta el NVR al internet vía cable ethernet, este viene con el kit).
  2. Encendido del NVR: Enciende el NVR y espera a que inicie completamente. El software del NVR se abrirá automáticamente.
  3. Configuración inicial: Al iniciar el software se abrirá una pestaña con opciones básicas, selecciona el idioma preferido, configura la fecha y la hora y elige los dispositivos a usar.
  4. Prueba del sistema: Realiza una prueba para asegurarte de que las cámaras están grabando y que puedes acceder a las imágenes en tiempo real a través del monitor.
  5. Finalización: Guarda todos los cambios realizados y asegúrate de que el sistema esté funcionando correctamente. Revisa periódicamente las grabaciones y ajusta la configuración según sea necesario.

Hasta ese punto, todo funcionará bajo parámetros predeterminados; las cámaras ya estarán activas, estarán con el sensor de movimiento activado y grabando automáticamente. Estos parámetros predeterminados siguen siendo muy básicos y limitan mucho el uso de las cámaras, estos parámetros se pueden modificar dentro de la configuración avanzada.

Configuración avanzada

  1. Acceso a la configuración avanzada: Para entrar a la configuración necesitarás hacer uso de la clave de administrador, esta se encuentra en el reverso de la caja. Una vez se use la contraseña, el mismo software te dará la opción de cambiar la contraseña y elegir un nombre de usuario.
  2. Configuración de parámetros: Para entrar a esta sección deberás hacer clic derecho en la pantalla y apretar la opción menú general. Dentro del menú habrá muchas especificaciones donde se puede configurar el sensor de movimiento, calidad de las cámaras, grabación de video, entre otros.
    1. Parámetros de la cámara: Una vez abierto el menú general, nos vamos a la opción "grabación". Una vez ahí, vamos a la sección que dice parámetros, donde nos aparecen múltiples opciones para cambiar, ya sea la calidad de video, resolución de la cámara, tasa de fotogramas, etc. Para modificar estas opciones, deberás hacer clic izquierdo sobre estos y te aparecerán múltiples opciones para cambiar a tu gusto. Ejemplo: cambiar la resolución de "1280*720 (HD)" a "2560*1440".
    2. Sensor de movimiento: Para activar o desactivar el sensor de movimiento, abrimos el menú general y apretamos la opción "cámara". Una vez ahí, apretamos la opción "movimiento", donde tendrás la opción de activar o desactivar el sensor y su sensibilidad. Estos parámetros son individuales para cada cámara.
  3. Aplicación: Todas las cámaras de Hikvision se pueden usar con la aplicación "Hik-connect". Para hacer esto, primero debes descargar la aplicación desde la Play Store o App Store. Luego, regístrate creando una cuenta utilizando tu número de teléfono o correo electrónico, e inicia sesión con tus credenciales. Una vez dentro, selecciona la opción para agregar dispositivos, ya sea escaneando el código QR que aparece en tu NVR o DVR, o ingresando manualmente el número de serie del dispositivo. Configura las opciones de visualización y notificaciones según tus preferencias, habilitando alertas de movimiento y ajustando la calidad de video. Finalmente, asegúrate de que tu dispositivo esté conectado a Internet para poder monitorear tus cámaras en tiempo real, reproducir grabaciones y recibir notificaciones de eventos desde cualquier lugar.
