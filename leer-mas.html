Requisitos

Para seguir este tutorial, asegúrate de tener instalados los siguientes programas:

Descargar Python Descargar Visual Studio Code

Descripción del Video

Este video te introduce al lenguaje de programación Python de manera sencilla y clara. Es perfecto para principiantes que quieren aprender a programar.

¿Qué vas a aprender?

Crear tu primer programa

Escribirás un programa que imprima "Hola mundo".

Ejemplo de Programa "Hola Mundo"

print("Hola, mundo")

Variables

Aprenderás a crear variables para almacenar datos. Por ejemplo, puedes almacenar tu edad en una variable:

Ejemplo de Variables

age = 25
print(f"Tienes {age} años.")

Operadores

Aprenderás a usar operadores básicos como suma, resta, multiplicación y división para realizar cálculos.

Ejemplo de Operadores

a = 10
b = 4

suma = a + b
resta = a - b
multiplicacion = a * b
division = a / b

Sentencias condicionales

Aprenderás a usar sentencias if para controlar el flujo del programa. Por ejemplo, puedes verificar si una persona es mayor de edad:

Ejemplo de Sentencias Condicionales

edad = 18
if edad >= 18:
    print("Eres mayor de edad.")
else:
    print("Eres menor de edad.")

Bucles

Aprenderás a usar bucles for y while para repetir código. Por ejemplo, puedes imprimir "Hola" tres veces:

Ejemplo de Bucles

count = 0
while count < 3:
    print("Hola")
    count += 1

Al finalizar el video, tendrás una comprensión básica de Python y estarás listo para comenzar a programar.
