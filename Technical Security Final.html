Technical Security

"Technical Security" es un grupo de estudiantes de cuarto medio que se especializó en el desarrollo de una casa domótica, enfocándose principalmente en la prevención de delitos como los portonazos. Este equipo se dedica a implementar soluciones tecnológicas innovadoras que mejoren la seguridad del hogar, utilizando dispositivos y sistemas que se pueden controlar a través de una aplicación propia diseñada para tal fin.

Casa Domotica

Una casa domótica es un hogar que integra tecnología avanzada para automatizar y controlar diversas funciones, mejorando la comodidad, seguridad y eficiencia energética. Estos sistemas permiten gestionar aspectos como la iluminación, climatización, seguridad y electrodomésticos a través de dispositivos conectados, que se comunican entre sí y con el usuario mediante aplicaciones móviles o asistentes de voz. La domótica no solo optimiza el consumo de energía, sino que también facilita la vida diaria al permitir la programación de tareas y la monitorización remota del hogar. Además, contribuye a un entorno más seguro, ya que incluye sistemas de alarma y vigilancia que pueden ser controlados de manera centralizada o distribuida. En resumen, una casa domótica representa una evolución significativa en la forma en que interactuamos con nuestro entorno, adaptándose a las necesidades y preferencias de sus habitantes.

Maqueta

Este proyecto cuenta con dos maqueta, una fisica y virtual. La maqueta Fisica simula una casa domótica incluyendo sus funcionamientos, esto se logra gracias a sus material como pueden ser el "Arduino Uno" (micro crontolador), un sensor ultrasonico, servo motor , Modulo Bluetooth, entre otros materiales. La maqueta virtual es un plano en 3D de una casa domotica en funcionamiento, el plano esta creado en un software de diseños casas en 3D, El softwere se llama "Cedreo".
